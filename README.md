# Wallet Club

> La plataforma de fidelización más simple para pequeños y medianos comercios.

## 🚀 Misión

Ayudar a los comercios a construir relaciones duraderas con sus clientes mediante tarjetas digitales compatibles con Apple Wallet y Google Wallet.

## 🎯 Visión

Convertirnos en la plataforma de fidelización más utilizada de Latinoamérica.
